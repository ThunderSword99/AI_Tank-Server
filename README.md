## Run server
### Install modules
#### :warning: Just installing in the first time you use

- Open terminal in Visual Studio Code (Ctrl+J) then type:

`npm install yarn`

`yarn`

`yarn add socket.io-client`

`yarn add config`

### Run game server
- Open terminal in Visual Studio Code (Ctrl+J) then type:

`node app`

### Run web server
- Open terminal in Visual Studio Code (Ctrl+J) then type:

`nodemon`
